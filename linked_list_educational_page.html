What is a Linked List?

A linked list is a linear data structure made up of individual units called nodes. Unlike arrays, the elements are not stored in contiguous memory. Instead, each node stores a reference (pointer) to the next node, forming a chain-like structure.

Key Idea: Each node knows where the next node lives in memory. The connection is logical, not physical.

Why use Linked Lists? When data changes frequently — elements are inserted or deleted often — linked lists outperform arrays because no element-shifting is needed.

Node Structure in Python

In Python, a node is represented as a class with two attributes: data (the value) and next (a reference to the next node).

# Basic Node Class
class Node:
    def __init__(self, data):
        self.data = data        # stores the value
        self.next = None       # pointer to next node (None = NULL)

# Creating nodes
n1 = Node(10)
n2 = Node(20)
n3 = Node(30)

# Linking nodes manually
n1.next = n2   # 10 → 20
n2.next = n3   # 20 → 30
# n3.next remains None → 30 → NULL

head = n1      # head points to the first node

3
Memory Representation

Nodes can reside anywhere in memory. The link (pointer) in each node bridges the physical gap. This is the fundamental difference from arrays.

FeatureArrayLinked List
Memory layoutContiguous blocksScattered anywhere
AccessDirect (index)Sequential (traverse)
SizeFixed at creationDynamic (grows/shrinks)
InsertionCostly (shift elements)Cheap (adjust pointers)

Advantages

  • Dynamic — no fixed size limit
  • Fast insertion & deletion
  • No memory wastage from preallocation
  • Base for stacks, queues, graphs

Disadvantages

  • No direct/random access
  • Extra memory for pointers
  • Slow sequential traversal
  • More complex to implement

Types of Linked Lists
4.1 Singly Linked List

Each node has one pointer — to the next node. Traversal is one-directional (forward only). The last node points to None.

Singly Linked List
HEAD
5
→
→
15
→
→
25
→
→
None
class SinglyLinkedList:
    def __init__(self):
        self.head = None

    def insert_at_beginning(self, data):
        new_node = Node(data)
        new_node.next = self.head  # point new node to old head
        self.head = new_node        # update head

    def insert_at_end(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        temp = self.head
        while temp.next:       # traverse to last node
            temp = temp.next
        temp.next = new_node   # attach new node

    def display(self):
        temp = self.head
        while temp:
            print(temp.data, end=" → ")
            temp = temp.next
        print("None")

Each node stores two pointers — to both the previous and next node. Enables bidirectional traversal. Used in browser history, undo/redo, music playlists.

class DNode:
    def __init__(self, data):
        self.data = data
        self.prev = None   # pointer to previous node
        self.next = None   # pointer to next node

class DoublyLinkedList:
    def __init__(self):
        self.head = None

    def insert_at_end(self, data):
        new_node = DNode(data)
        if not self.head:
            self.head = new_node
            return
        temp = self.head
        while temp.next:
            temp = temp.next
        temp.next = new_node   # link forward
        new_node.prev = temp   # link backward

    def display_forward(self):
        temp = self.head
        while temp:
            print(temp.data, end=" ⇄ ")
            temp = temp.next

    def display_backward(self):
        temp = self.head
        while temp.next:       # go to last node
            temp = temp.next
        while temp:            # traverse backwards
            print(temp.data, end=" ⇄ ")
            temp = temp.prev

4.3 Circular Singly Linked List

The last node's next pointer points back to the head instead of None. There is no null terminator. Great for round-robin scheduling and cyclic applications.

Circular Singly — last node links back to head
HEAD
1
→
→
2
→

3
↩
↩ Node 3's next → points back to Node 1 (HEAD)
class CircularSinglyLinkedList:
    def __init__(self):
        self.head = None

    def insert_at_end(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            new_node.next = self.head   # point to itself
            return
        temp = self.head
        while temp.next != self.head:  # stop at last node
            temp = temp.next
        temp.next = new_node
        new_node.next = self.head       # close the circle

    def display(self):
        if not self.head: return
        temp = self.head
        while True:
            print(temp.data, end=" → ")
            temp = temp.next
            if temp == self.head:      # back to start — stop!
                break
        print("(head)")
Caution: Always check temp != head as the loop stop condition in circular lists, or you risk an infinite loop.
4.4 Circular Doubly Linked List

Combines doubly-linked and circular properties. The last node's next → head and the head's prev → last node. Most powerful — supports bidirectional traversal in a loop.

class CircularDoublyLinkedList:
    def __init__(self):
        self.head = None

    def insert_at_end(self, data):
        new_node = DNode(data)
        if not self.head:
            self.head = new_node
            new_node.next = new_node    # self-loop (next)
            new_node.prev = new_node    # self-loop (prev)
            return
        last = self.head.prev          # last is head's prev in circular
        last.next = new_node
        new_node.prev = last
        new_node.next = self.head      # close forward loop
        self.head.prev = new_node       # close backward loop

Traversal — Visit every node from head to the end. The foundation of all other operations. Time: O(n)
1
Set temp = head
2
While temp is not None, visit (print/process) temp.data
3
Move: temp = temp.next
4
Repeat until temp becomes None
def traverse(self):
    temp = self.head
    count = 0
    while temp:
        print(f"Node {count}: {temp.data}")
        temp = temp.next
        count += 1
    print(f"Total nodes: {count}")
Insertion — Add a new node at beginning O(1), end O(n), or a specific position O(n).
# --- Insert at Beginning: O(1) ---
def insert_beginning(self, data):
    new_node = Node(data)
    new_node.next = self.head  # step 1: connect to old head
    self.head = new_node        # step 2: new node IS head now

# --- Insert at End: O(n) ---
def insert_end(self, data):
    new_node = Node(data)
    if not self.head:
        self.head = new_node
        return
    temp = self.head
    while temp.next:
        temp = temp.next        # reach last node
    temp.next = new_node        # attach

# --- Insert at Position: O(n) ---
def insert_at_position(self, data, pos):
    if pos == 0:
        self.insert_beginning(data)
        return
    new_node = Node(data)
    temp = self.head
    for _ in range(pos - 1):
        if not temp.next:
            raise IndexError("Position out of bounds")
        temp = temp.next
    new_node.next = temp.next   # bridge: new → what was next
    temp.next = new_node        # bridge: prev → new

# --- Insert After a Given Node ---
def insert_after_node(self, target_data, new_data):
    temp = self.head
    while temp:
        if temp.data == target_data:
            new_node = Node(new_data)
            new_node.next = temp.next
            temp.next = new_node
            return
        temp = temp.next
    print("Target not found")

Deletion — Remove a node by position or value. No element shifting needed — just update pointers. Beginning: O(1), others: O(n).
# --- Delete from Beginning: O(1) ---
def delete_beginning(self):
    if not self.head:
        return
    self.head = self.head.next    # move head forward

# --- Delete from End: O(n) ---
def delete_end(self):
    if not self.head: return
    if not self.head.next:
        self.head = None   # only one node
        return
    temp = self.head
    while temp.next.next:     # reach second-to-last
        temp = temp.next
    temp.next = None          # cut off last node

# --- Delete by Value: O(n) ---
def delete_by_value(self, key):
    if not self.head: return
    if self.head.data == key:
        self.head = self.head.next
        return
    temp = self.head
    while temp.next:
        if temp.next.data == key:
            temp.next = temp.next.next  # bypass the node
            return
        temp = temp.next
    print("Value not found")

Searching — Linear scan from head, compare each node's data. Time: O(n). No random access unlike arrays.
# --- Linear Search ---
def search(self, key):
    temp = self.head
    position = 0
    while temp:
        if temp.data == key:
            print(f"Found {key} at position {position}")
            return position
        temp = temp.next
        position += 1
    print(f"{key} not found in list")
    return -1

# --- Count nodes ---
def count(self):
    temp = self.head
    count = 0
    while temp:
        count += 1
        temp = temp.next
    return count  # O(n)
Reversal — Flip all pointers. Uses 3 pointer variables. Time: O(n). Space: O(1)
1
Initialize: prev = None, current = head, next_node = None
2
While current: save next, flip current's pointer backward, advance both pointers
3
At end: head = prev (prev is now the last-visited = first of reversed)
# --- Iterative Reversal: O(n) time, O(1) space ---
def reverse(self):
    prev = None
    current = self.head
    while current:
        next_node = current.next    # save next
        current.next = prev         # reverse pointer
        prev = current              # advance prev
        current = next_node         # advance current
    self.head = prev               # prev is new head

# --- Recursive Reversal: O(n) time, O(n) stack space ---
def reverse_recursive(self, node):
    if not node or not node.next:
        self.head = node
        return
    self.reverse_recursive(node.next)
    node.next.next = node
    node.next = None

Sorting — Bubble sort is easiest to implement on linked lists. Merge sort is optimal O(n log n) since it doesn't need random access.
# --- Bubble Sort on Linked List: O(n²) ---
def bubble_sort(self):
    if not self.head: return
    swapped = True
    while swapped:
        swapped = False
        temp = self.head
        while temp.next:
            if temp.data > temp.next.data:
                # swap data, not nodes (simpler)
                temp.data, temp.next.data = temp.next.data, temp.data
                swapped = True
            temp = temp.next

# --- Merge Sort on Linked List: O(n log n) --- PREFERRED
def merge_sort(head):

        return head
    mid = get_middle(head)

    mid.next = None           # split into two halves
    left = merge_sort(head)

def get_middle(head):
    slow = fast = head
    while fast.next and fast.next.next:
        slow = slow.next             # slow: 1 step
        fast = fast.next.next        # fast: 2 steps → slow is mid
    return slow

def sorted_merge(a, b):
    if not a: return b
    if not b: return a
    if a.data <= b.data:
        a.next = sorted_merge(a.next, b)
        return a
    else:
        b.next = sorted_merge(a, b.next)
        return b

Concatenation & Splitting — Join two lists by linking the tail of the first to the head of the second. Splitting is the reverse — cut at a midpoint.
# --- Concatenation: O(n) ---
def concatenate(list1_head, list2_head):
    if not list1_head: return list2_head
    temp = list1_head
    while temp.next:           # find last node of list1
        temp = temp.next
    temp.next = list2_head     # link to head of list2
    return list1_head

# Result: 1→2→3→4→5→6 (list1 + list2 joined)

# --- Split into Two Halves (slow-fast pointer) ---
def split_list(head):
    if not head or not head.next:
        return head, None
    slow = head
    fast = head.next
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    second_half = slow.next
    slow.next = None       # cut the list
    return head, second_half

Floyd's Cycle Detection (Detect Loop)

Use two pointers — slow (1 step) and fast (2 steps). If they ever meet, a cycle exists.

🎨 New Image Prompt — Floyd's Cycle Detection

# --- Floyd's Cycle Detection: O(n) time, O(1) space ---
def detect_cycle(self):
    slow = self.head
    fast = self.head
    while fast and fast.next:
        slow = slow.next         # 1 step
        fast = fast.next.next    # 2 steps
        if slow == fast:
            print("Cycle DETECTED!")
            return True
    print("No cycle.")
    return False

# --- Find Start of Cycle ---
def find_cycle_start(self):
    slow = fast = self.head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            slow = self.head   # reset slow to head
            while slow != fast:
                slow = slow.next
                fast = fast.next
            return slow.data   # cycle start node
    return None
Find Middle Element (Slow-Fast Pointer)
def find_middle(self):
    slow = fast = self.head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next  # when fast reaches end, slow is at middle
    return slow.data

Check if Palindrome
def is_palindrome(self):
    # Collect all values
    values = []
    temp = self.head
    while temp:
        values.append(temp.data)
        temp = temp.next
    return values == values[::-1]  # compare with reversed

# Optimal approach: reverse second half and compare in-place (O(1) space)
def is_palindrome_optimal(self):
    if not self.head or not self.head.next: return True
    slow = fast = self.head
    while fast.next and fast.next.next:
        slow = slow.next
        fast = fast.next.next
    # Reverse second half
    second_start = slow.next
    prev = None
    while second_start:
        nxt = second_start.next
        second_start.next = prev
        prev = second_start
        second_start = nxt
    # Compare
    left, right = self.head, prev
    while right:
        if left.data != right.data: return False
        left = left.next
        right = right.next
    return True
Nth Node from End (Two Pointer)
def nth_from_end(self, n):
    """Find the Nth node from the end — one pass, two pointers"""
    first = second = self.head
    for _ in range(n):
        if not first:
            raise ValueError("N is larger than list length")
        first = first.next     # advance first by N steps
    while first:
        first = first.next
        second = second.next   # both advance together
    return second.data         # second is now N from end

Intersection of Two Linked Lists
def find_intersection(head1, head2):
    """Find where two linked lists merge — O(n+m) time, O(1) space"""
    a, b = head1, head2
    while a != b:
        a = a.next if a else head2  # switch to other list at end
        b = b.next if b else head1
    return a  # None if no intersection, else the node
